Detailed Description

Abstract Subject class representing a sensor in the smart home system.

The Sensor class serves as an abstract base class for sensors that monitor environmental conditions and notify registered smart devices about changes. It manages a list of SmartDevice instances that are interested in receiving notifications when sensor data changes.
