\doxysubsection{Detailed Description}
Represents a smart door lock device in the smart home system. 

The \doxylink{class_door_lock}{Door\+Lock} class extends the \doxylink{class_smart_device}{Smart\+Device} class and includes functionality specific to a door lock, such as locking and unlocking the door, as well as reporting its current state.
