\doxysubsection{Detailed Description}
Represents a motion sensor in the smart home system. 

The \doxylink{class_motion_sensor}{Motion\+Sensor} class extends the \doxylink{class_sensor}{Sensor} class and implements functionality to detect motion and notify registered smart devices about the detected motion events.
