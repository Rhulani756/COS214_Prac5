\doxysubsection{Detailed Description}
Abstract Subject class representing a sensor in the smart home system. 

The \doxylink{class_sensor}{Sensor} class serves as an abstract base class for sensors that monitor environmental conditions and notify registered smart devices about changes. It manages a list of {\ttfamily \doxylink{class_smart_device}{Smart\+Device}} instances that are interested in receiving notifications when sensor data changes.
