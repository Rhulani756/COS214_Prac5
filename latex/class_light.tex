\doxysubsection{Detailed Description}
Represents a smart light device in the smart home system. 

The \doxylink{class_light}{Light} class extends the \doxylink{class_smart_device}{Smart\+Device} class and implements methods specific to a light device, such as turning the light on/off and checking its status.
