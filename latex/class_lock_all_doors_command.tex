\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} class to lock all doors in a specified room. 

This class implements the \doxylink{class_command}{Command} interface, providing a concrete command for locking all doors within a specified room. The command encapsulates the action of locking doors, allowing it to be executed within the context of a macro routine or directly by a command invoker.
