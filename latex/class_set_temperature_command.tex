\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} class to set the temperature in a specified room. 

This class implements the \doxylink{class_command}{Command} interface, providing a concrete command for setting the temperature in a specified room. The command encapsulates the action of adjusting the temperature, allowing it to be executed within the context of a macro routine or directly by a command invoker.
