\doxysubsection{Detailed Description}
An abstract base class for all smart devices in the smart home system. 

The \doxylink{class_smart_device}{Smart\+Device} class extends the \doxylink{class_smart_component}{Smart\+Component} class and defines additional functionality specific to smart devices. It includes methods for performing actions, retrieving the device\textquotesingle{}s status, and getting the type of the device.
